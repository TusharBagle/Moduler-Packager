# Multi-Utility Toolkit

## About
Multi-Utility Toolkit is a Python-based application designed to perform multiple utility operations through an interactive menu-driven interface.

This project combines important Python concepts such as Object-Oriented Programming, datetime handling, mathematical calculations, random data generation, file handling, UUID generation, and module exploration while improving logical thinking and practical programming skills.

## Features
- Display current date and time
- Calculate difference between two dates
- Stopwatch and countdown timer
- Factorial and compound interest calculation
- Trigonometric calculations
- Area calculation for geometric shapes
- Generate random numbers, lists, passwords, and OTPs
- Generate UUIDs
- Create, read, write, and append files
- Explore module attributes using dir()

## Concepts Used
- Classes & Objects
- Inheritance
- File Handling
- Exception Handling
- Datetime Module
- Math Module
- Random Module
- UUID Module
- String Module
- Menu-Driven Programming

## Technologies Used
- Python

## Purpose
The main purpose of this project is to strengthen understanding of Python built-in modules, Object-Oriented Programming concepts, and utility-based application development through practical implementation.
